各種明細のPDFに閲覧パスワードが設定されていることがあります. それぞれ, なんらかの理由があって設定されているはずです. しかしながら, 単に個人のコンピュータで, 個人的に閲覧しようとするたびに (一般的なビューワで) 毎回パスワードを問われるのも面倒だと思います.

比較的簡単に, 毎度のパスワード入力をせずに済む方法がわかったのでここに書いておきます. 当然ながら, パスワードを解除してよい書類にのみ使用し, そうでない書類でこれを使用すべきではありません. もしこのページの内容を用いて何かをする場合, 個人の判断で行ってください. 私は一切の責任を負いません.

QPDFを使う

QPDF を使うと良さそうです. 導入方法は様々です. macOSの場合, Homebrew からインストールできます.

brew install qpdf

使い方

解除したい original.pdf を用意し, 出力したいファイル名を clean.pdf とするならば, 次のコマンドを実行します.

qpdf --decrypt --password=**** original.pdf clean.pdf

これだけで完了です.

他の方法として, 仮想プリンタに印刷する方法や, (macOSの場合) プリントダイアログから「PDF」ポップアップメニューを使用してPDFファイルを作成する方法もありますが, これらはPDFの外観は似ていても内部の構造は恐らく変わってしまいますし, 操作の量も多いです.

QPDFはこれらよりも少ない手順で, ターミナルから簡単に処理できるので良いかと思います.

ただし, コマンドラインに直接パスワードを打ち込むと, .zsh_history や .bash_history にパスワードが残ってしまう場合があるので, 重要なパスワードを扱う場合は工夫が必要です.
