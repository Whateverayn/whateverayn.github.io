= 波の速さ

- 波の速さ: $v space [upright(m)\/upright(s)], quad$ 波長: $lambda space [upright(m)], quad$ 周期: $T space [upright(s)], quad$ 周波数: $f space [upright(H z)]$

$ v=lambda/T=f lambda $

= マクスウェル方程式

== 電場に関するガウスの法則

- 湧き出す電束密度 $bold(D)$ は電荷 $rho$ に等しい

$ bold(nabla) dot bold(D) = "div" bold(D) = rho $

== 磁場に関するガウスの法則

- 磁束密度 $bold(B)$ は湧き出さない (増減しない)

$ bold(nabla) dot bold(B) = "div" bold(B) = 0 $

== ファラデーの電磁誘導の法則

- 時間変化する磁束密度 $bold(B)$ は, 周囲に渦状の電場 $bold(E)$ を生成する

$ bold(nabla) times bold(E) = "rot" bold(E) = - (partial bold(B))/(partial t) $

== アンペール・マクスウェルの法則

- 磁場の強さ $bold(H)$ は, 電流密度 $bold(J)$ と時間変化する電場 $bold(E)$ によって生成される

$ bold(nabla) times bold(H) = "rot" bold(H) = bold(J)+(partial bold(D))/(partial t) $

== ポインティングベクトル $(bold(S) = bold(E) times bold(H))$

- 単位時間, 単位面積当たりに流れる電磁エネルギーを表すベクトル量

$
  S_n = - E H, quad E = R I, quad H = I/(2 pi r)\
  P = integral_S_1 (- E H) d S = - (R I^2)/(2 pi r) dot 2 pi r dot 1 = - R I^2 space [upright(W)]
$

= 波動方程式

$ bold(nabla)^2 bold(E) = epsilon mu (partial^2 bold(E))/(partial t^2), quad bold(nabla)^2 bold(H) = epsilon mu (partial^2 bold(H))/(partial t^2) $

= 真空の光速度

$ c = 1/(sqrt(epsilon_0 mu_0)) space [upright(m)\/upright(s)] $

= 特性インピーダンス $(eta)$

- 真空: $377 ohm$

$ E/H = eta = sqrt(mu/epsilon) space[upright(ohm)] $

= 境界条件

- $D_(1 upright(n)) = D_(2 upright(n))$: *誘電率*の異なる界面では, *電束密度* $bold(D)$ の境界面に*垂直*な成分が保存される
- $E_(1 upright(t)) = E_(2 upright(t))$: *誘電率*の異なる界面では, *電場* $bold(E)$ の境界面に*平行*な成分が保存される
- $B_(1 upright(n)) = B_(2 upright(n))$: *透磁率*の異なる界面では, *磁束密度* $bold(B)$ の境界面に*垂直*な成分が保存される
- $H_(1 upright(t)) = H_(2 upright(t))$: *透磁率*の異なる界面では, *磁場の強さ* $bold(H)$ の境界面に*平行*な成分が保存される
